# exfe_cli

Command line interface to [exfe.com](https://exfe.com/)

## Installation

    gem install exfe

Compatible with Ruby 1.8.7+ and Ruby 1.9+

## Usage

List my events

    exfe events

View an event

    exfe event EVENT_ID

Comment on an event

    exfe comment EVENT_ID

View a user

    exfe user USER_ID
