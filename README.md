# exfe_cli

Command line interface to [exfe.com](https://exfe.com/)

## Installation

    gem install exfe

Compatible with Ruby 1.8.7+ and Ruby 1.9+

## Usage

List my crosses

    exfe crosses

View a cross

    exfe cross CROSS_ID

Comment on a cross

    exfe comment CROSS_ID

View my profile

    exfe profile

View a user

    exfe user USER_ID
