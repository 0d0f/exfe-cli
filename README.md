# exfe-cli

Command line interface to [exfe.com](https://exfe.com/)

## Installation

    [sudo] gem install exfe

Install from source code

    [sudo] gem install bundler && bundle && rake build && [sudo] rake install

Compatible with Ruby 1.8.7+ and Ruby 1.9+

## Usage

List my crosses

    exfe crosses

View a cross

    exfe cross CROSS_ID
